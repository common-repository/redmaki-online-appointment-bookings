== Description ==

If you use [Redmaki](https://www.redmaki.com/) for appointment scheduling on your website, you can embed your appointment widget on your site using this plugin.
All you need is your api key which you can find by loging into your account.

Features
* Ultra easy setup
* Supports the language you selected in Redmaki
* Fully integrates in your Wordpress website

If you don't have a Redmaki account yet, you can register on [our website](https://www.redmaki.com).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Redmaki screen to activate the plugin and set your api key
1. Check out your website with the Redmaki Book Now button

== Frequently Asked Questions ==

= Is it free? =

This plugin is completely free. Redmaki has a free trial period of 1 month which you can stop any time you want.

= Who uses Redmaki? =

Redmaki can be used by any company which takes bookings from customers. Redmaki is used by hairdressers, beauty or nail salons, docters, opticians, tattoo shops, etc.
